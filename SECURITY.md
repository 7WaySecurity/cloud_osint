## Reporting a Vulnerability

If you discover a security issue related to this repository (for example, a resource that links to a compromised or malicious site), please report it responsibly.

**Email:** giovanni.cruz@7waysecurity.co

**What to include:**

- Description of the issue
- Which entry or link is affected
- Steps to reproduce or verify
- Any additional context

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Assessment:** Within 7 days
- **Resolution:** Depending on severity, typically within 14 days

## Scope

This security policy covers:

- Links in the repository that may have been compromised or redirected to malicious content
- Resources that have been deprecated and now serve harmful content
- Any content in this repository that could inadvertently cause harm

## Responsible Disclosure

We kindly ask that you:

- Give us reasonable time to address the issue before public disclosure
- Act in good faith to avoid privacy violations, data destruction, or service interruption
- Do not exploit any vulnerabilities you discover
